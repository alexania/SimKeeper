Get Started

Start your records either by:

  • Adding your first Sim:
    • Click the "Add Sim" button or press enter.
    • The button will be disabled the name is less than 3 characters long.
    • This text box doubles as search!
    • The first Sim added will be set as the Root Sim and the family name will be set to its surname. (This can be changed afterward, don't stress.)
  • Import a GEDCom file
    • You can use MCCC to generate a GedCom.
    • This will overwrite the current records.
    • The MCCC GedCom file doesn't have a concept of dates or sequence of events, so those won't be imported.
    • It also assumes that all parents were married at some point, so you might have to go delete some marriages.
    • If you're using a GedCom that contains info I'm not using, send it to me and I'll look into updating the import.

Global Configuration

Things you can edit:

  • The Global Age Spans - this is used to calculate the current stage, it can be overwritten on an individual basis.
  • The Family Name - it will take the first Sim's surname by default but can be changed at any time.
  • The current day - either click the + to increment it or edit it manually.

Navigation

  • Collapse and Expand all records - Be careful, expanding all on large lists will slow things way down.
  • Find - start typing to search for a Sim, click "Next" to jump to the next found instance... it's a search bar.
  • Collapse and Expand individual records.

Save and Load

This site doesn't currently support any form of account or data storage, as such your records needs to be stored locally.
  • Save and Load - You can save your records to download them as a JSON file. To continue where you left off you can just load that same JSON file.

Editing a Sim


You can change the following things on an individual record (just click on the field you want to edit unless otherwise specified).
  • Unique ID - This needs to be unique.
  • Name
  • Birthday - This can either be changed via the Birthday field or the Birth event under history.
  • Stage - Change this if you don't want the stage to be auto calculated.
  • Traits - Click to select / deselect Sim traits.
  • Career
  • Place
  • History
    • Date - Click the date value to change it.
    • Event Details - Click the "Day" text on an event to change other information about it.
    • Delete Event - If the event is deletable, you can click the red cross to delete it.
    • Add events - click the appropriate button to add an event.
  • Delete Sim - Permanently delete this sim. All references to it will be changed to "Unknown".
  • Configure Sim
    • Root Sim - Click to set this Sim as the root sim. The root sim will always be at the top of the list.
    • Completed - You can check this to mark this Sim as completed (all info is filled in), this will colour the record green.
    • Age Spans - You can override the Global Age Spans (see Global Configuration) for an individual sim. Click to open/close.

Frequently Asked Questions

How do I know what day it is in game?
I haven't actually found a way to do this. Since I started these records mid-game I guesstimated it based on the age spans (I use the Sim Info mod to work out the sim's age).
If you're starting the records off at the beginning it's easier so long as you remember to increment the day every in-game midnight. If you don't want to use the "Day" functionality, just ignore it, override the stages and use it to order events.
Can I change the Sim portraits?
For the moment no. Since I'm not storing any info there's no real way of uploading images.
Two ways I can implement this is to either provide a selection of preset images to choose from, or allow you to provide a url to an image hosted somewhere else. Let me know which method you'd prefer.

In all seriousness, I created this site for my own personal use. So I made it work in Chrome. Right now, I don't have any immediate plans to test it in a myriad of other browsers and all their future updates.
